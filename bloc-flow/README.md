| ---------------------------- | --------------------------- |

| `BlocFlow: New Mode`         | Generate a new Mode         |
| `BlocFlow: New Presentation` | Generate a new Presentation |
| `BlocFlow: New Bloc`         | Generate a new Bloc         |
| `BlocFlow: New Feature`      | Generate a new Feature      |
